## The World’s Living Archive of Legendary Chef Collaborations + A Community Collab Engine

# Executive Vision

**ChefCollabs.com** is not a food news blog.

It is a **living global repository** of chef collaborations — past, present, and emerging — and the first platform that turns chef collabs into a learnable, repeatable craft for everyday communities.

ChefCollabs.com becomes:

- the world archive of chef collabs (pop-ups, takeovers, co-menus, residencies, events, charity dinners, TV specials)
- a “how they built it” cookbook + playbook
- a technique-to-dish recreation library
- a local collab matchmaking engine for friends/neighborhood chefs
- a global directory of top culinary schools in major cities as community resources

This is where collaboration becomes a culinary movement.

# 4) The Collab Archive (Crown Jewel)

ChefCollabs is a **living archive** — it grows forever.

## Collab Page Template (Core Content Unit)

Every collab entry includes:

1) **Who** collaborated
- chef names + roles
- restaurants/affiliations (at the time)
- cuisine backgrounds

2) **What** they did
- pop-up / takeover / co-menu / residency / TV / charity dinner / brand activation
- location + venue
- date range (or year)
- theme and intent

3) **What they made**
- menu items (structured list)
- signature dish highlights
- techniques featured

4) **How to try to make it**
Two modes:
- **Home Mode** (realistic ingredients, simplified technique)
- **Chef Mode** (full technique, plating, timing)

5) **Chef DNA Notes**
- what each chef contributed (technique, flavor profile, plating, ingredients, philosophy)

6) **Pairing Suggestions**
- “If you like this collab, try these similar pairings”
- local pairing ideas (see Collab Engine)

7) **Media Anchors**
- YouTube interviews, event videos, chef demos when available
- press references (citations on the live site)

> Important: The archive should be positioned as “largest living library,” not “complete list of every collab ever.”

# 5) The “Make It” Library (Recreation Engine)

This is what turns ChefCollabs into a **masterpiece**, not a directory.

## Dish Recreation Template

Each recreated dish includes:
- ingredients list (grams + household measures)
- step-by-step directions
- timing checkpoints
- substitutions by geography/budget
- “chef-level upgrade” options
- plating notes
- pairing notes (wine, NA, dessert, side)

Add:
- technique links to the encyclopedia (sauces, emulsions, fermentation, grilling, pastry, etc.)

# 6) The Collab Engine (Community Feature)

ChefCollabs.com includes an interactive tool:

## “Pair Chefs Like a Producer”

Users input:
- cuisines (primary + secondary)
- skill level(s)
- vibe (fine dining / street food / comfort / experimental / family style)
- event type (dinner party / fundraiser / pop-up / tasting menu / picnic)
- dietary constraints
- equipment available
- budget per guest

Outputs:
- 10 collaboration concepts with:
  - theme name
  - 5-course structure (or small plates)
  - division of labor (who owns what)
  - shared techniques
  - shopping plan
  - prep timeline
  - plating plan
  - playlist / ambiance suggestion (optional)

### “Chef Pairing Archetypes” (Examples)
- Fire + Sauce (grill master + saucier)
- Classic + Modernist (French technique + experimental)
- Sweet + Savory (pastry chef + savory chef)
- Street + Michelin (street food genius + fine dining plating)
- Spice + Smoke (South Asian spice + BBQ pitmaster)
- Ocean + Garden (seafood chef + vegetarian chef)
- Heritage + Fusion (traditional + diaspora remix)

This makes collaboration learnable for ordinary people.

# 13) Trust, Accuracy, and Community Standards

ChefCollabs.com must:

- moderate community content firmly
